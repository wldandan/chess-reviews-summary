   - 推荐着法：`11...Kh8` 巩固王的位置

2. **12...Bd4** — 💥 严重失误
   - 原着法：`12...Bd4`
   - 推荐着法：`12...Be6` 保持象的协调
   - 原因：象走到 d4 被白方 Nxd4 强制兑换，黑方子力协调性被破坏，局面从 -1.25 跌至 -2.18

3. **14...Nc6** — 💥 严重失误
   - 原着法：`14...Nc6`
   - 推荐着法：`14...Be6` 巩固中心
   - 原因：马跳 c6 让白格象进入 g4 位，白方子力活跃度大幅提升，局面从 -2.16 跌至 -2.40

4. **24...Qd7** — ⚡ 一般错误（但总体正确）
   - 原着法：`24...Qd7`
   - 推荐着法：`24...Kg7` 简化局面
   - 原因：虽然 Qd7 是全场最佳（将局面拉回均衡），但面对白 Ng6 威胁，Kg7 逃王更稳健

5. **26...dxe5** — ⚡ 一般错误
   - 原着法：`26...dxe5`
   - 推荐着法：`26...Be6` 保持中心控制
